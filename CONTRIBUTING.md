# Contributing

Use Python 3.12 and `uv sync --all-extras --dev`. Keep source, canonical domain, matching, planning, and destination transport separate. Use fictional minimized fixtures and never commit credentials or real portfolios. Run CI-equivalent checks before proposing changes.

Use Conventional Commits (`feat:`, `fix:`, `docs:`, `test:`, `chore:`). Reviews prioritize identity correctness, completeness, idempotency, privacy, and destructive-operation safeguards. Recommended branch protection: require pull requests, one approval, passing CI, conversation resolution, and prohibit force pushes to the default branch.
