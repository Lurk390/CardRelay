# Collectr browser research checklist

Determine the full portfolio page, pagination or infinite-scroll behavior, structured response completeness, embedded data, visible totals, variant/quantity representation, authentication redirects, and schema-change signals. Capture fictional sanitized fixture shapes only. Extraction must fail closed unless all pages/batches and total checks establish `complete` status.

No authoritative Collectr web-portfolio URL has yet been verified. The visible research harness therefore defaults to the official marketing site and accepts an explicit user-supplied `--url`. Inspection records only response counts and status classes—never URLs, headers, cookies, bodies, or collection data.
