# Integrations

Browser ingestion is necessary for free users and will use a visible persistent user-controlled session. Structured responses are preferred, then embedded data, with DOM extraction last. Users must comply with platform terms. No integration may bypass access controls, CAPTCHAs, rate limits, or anti-bot protections.
